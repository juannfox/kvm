# kvm

*Kubectl Version Manager*: Seamlessly switch between multiple `kubectl` versions.

![KVM by Dalle-2](./img/kvm.png)

KVM by Dalle-2

## TL;DR

### Install

```python
pip install .
python3 -m kvm version
```

### Use

```python
python3 -m kvm download
```
